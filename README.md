# NEON TODO - 科技待辦系統

這是一個升級版的科技感桌面待辦清單，具備精緻的火箭動畫通知功能。

## 🚀 升級功能
- **科技感 UI**：採用 Cyberpunk 風格的霓虹配色（Neon Blue）與深色背景。
- **編輯功能**：點擊 `EDIT` 按鈕即可修改任務內容與設定提醒時間。
- **火箭動畫通知**：
  - 當設定的提醒時間到達時，一個精緻的**火箭**會從螢幕左側飛入正中央。
  - 通知視窗會置於**所有視窗的最上層**，確保您不會錯過。
  - 點擊「我知道了」後，火箭會優雅地向右滑動並消失。
- **後台監控**：程式會自動在後台檢查時間，即使您正在處理其他事務，通知也會準時出現。

## 🛠️ 使用說明
1. **設定提醒**：點擊任務旁邊的 `EDIT`，在時間欄位輸入 `HH:MM` 格式（例如 `14:30`）。
2. **啟動通知**：確保程式正在執行，當時間到達時，火箭就會飛出來！
3. **介面操作**：
   - `DEPLOY`: 新增任務。
   - `EDIT`: 編輯任務與時間。
   - `DEL`: 刪除任務。

## 執行方式
直接執行目錄下的 `NEON_TODO.exe` 即可。
